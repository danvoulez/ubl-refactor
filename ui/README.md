# UBL Premium UI (Prototype)

Interface estática de alta fidelidade para:

- Chat com agente "chefe".
- Observabilidade com métricas e incidentes.
- Configurações avançadas de AI, telemetria e segurança.

## Como executar

```bash
cd ui
python3 -m http.server 4173
```

Abra `http://localhost:4173`.
