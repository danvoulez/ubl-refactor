# Architecture Decision Records (ADR)

Use ADRs for decisions that change architecture, trust model, rollout strategy, or operational guarantees.

## Format

- File name: `NNNN-title.md` (e.g., `0001-DOC-GOVERNANCE.md`)
- Sections:
  1. Context
  2. Decision
  3. Consequences
  4. Status (`proposed`, `accepted`, `superseded`)
