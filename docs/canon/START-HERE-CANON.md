# START-HERE-CANON

Canonical quickstart for operators and contributors.

For LLM-first onboarding, start here:
- `docs/START-HERE-LLM-FIRST.md`

Then validate non-negotiables in root:
- `START-HERE.md`

## Canon Entry Points

1. `START-HERE.md` (root, declarative non-negotiables)
2. `docs/canon/CANON-REFERENCE.md` (exhaustive canon registry)
3. `docs/canon/UNC-1.md` (numeric canon spec)
4. `schemas/unc-1.schema.json` (machine-readable numeric contract)
5. `docs/vm/OPCODES_NUM.md` (VM numeric opcode canon)
6. `ARCHITECTURE.md` (normative architecture + determinism contracts)

## Canon Drift Warning

If behavior changes in code without corresponding updates in canon docs/schema/tests,
assume drift until proven otherwise.
